Hipervinculo

También llamado vínculo, hiperenlace, es una referencia en un documento de hipertexto a otro documento o recurso. Combinado con una red de datos y un protocolo de acceso, se puede utilizar para acceder al recurso referenciado. El cual se puede guardar, ver, o mostrar como parte del documento referenciado.
