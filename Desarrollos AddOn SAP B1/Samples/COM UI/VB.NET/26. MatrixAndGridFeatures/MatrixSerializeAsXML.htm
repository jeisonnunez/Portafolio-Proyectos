SerializeAsXML()

 

Handling matrix objects is a common action done in add-ons.

So far, reading values from system matrix objects could only been done cell by cell in the GUI level or by reading from DataSource objects.

This way of getting information is slow, and in matrix objects with many lines it becomes a performance issue.

 

The SerializeAsXML function returns an XML document containing all the matrix data.

This saves the time of calling many functions when reading a matrix cell by cell; the performance using this method is improved dramatically.

 

Another benefit of this function is that the XML that is returned from this function is compatible with .NET objects, and can be used easily in .NET projects.

 

How to use the SerializeAsXML feature in .NET?

 

Step 1: Get the matrix schema and save it to a file.

 

string schema = matrix.GetSchema();

 

Step 2: Use the XML Schema Definition Tool (Xsd.exe).

 

In order to generate a class that corresponds to the matrix schema, we will use the XSD tool.

 

The command line should look like this: XSD.exe /classes MatrixSchema.xsd

 

The output of this command is a class in c# that has all the properties of the matrix.

This class can be used in .NET projects.

 

We can also use a different command: XSD.exe /dataset MatrixSchema.xsd that

generates a class derived from DataSet that corresponds to the matrix schema.

 

Step 3: Use the class in a .NET project.

 

First we get the matrix XML; here we do it from a marketing document form.

 

SAPbouiCOM.Matrix matrix;

SAPbouiCOM.Form form = oCon.Application.Forms.ActiveForm;

SAPbouiCOM.Item item = form.Items.Item("38");

matrix = (SAPbouiCOM.Matrix)item.Specific;

           

string MatrixXml = matrix.SerializeAsXML(SAPbouiCOM.BoMatrixXmlSelect.mxs_All);

 

Second, we use the class we created in Step 2.

We create an instance of the class and load a matrix into it.

 

Matrix objMatrix;

      // Construct an instance of the XmlSerializer with the type

      // of object that is being deserialized.

XmlSerializer mySerializer = new XmlSerializer(typeof(Matrix));

      // To read the file, create a FileStream.

System.IO.FileStream myFileStream = new System.IO.FileStream(@"C:\Debug\SerializeAsXML\matrix1.xml", System.IO.FileMode.Open);

      // Call the Deserialize method and cast to the object type.

      objMatrix = (Matrix)mySerializer.Deserialize(myFileStream);

 

Now, we can use this object like any other .NET object.
