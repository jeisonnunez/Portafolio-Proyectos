DI Server Sample

 

Sample requirements:

You need the following software installed:

  1. IIS Server.
  2. .Net framework
  3. Visual Studio .Net 2003

 

In order to use this sample follow these steps:

  1. Copy the folder “DISSample” (it’s in the same folder with this file) to your IIS folder, usually “C:\interpub\wwwroot”.
  2. Open the “Internet Services Manager” under Start-> Programs-> Administrative tools.
  3. Open your server node, and the default web server node.
  4. Locate the “DISSample” folder, right-click on it and select properties.
  5. Click the “Create” button circled in the following image.

 

 

  1. Open the file “DISSample.vbproj”
  2. Build the project and run it.
  3. Open the Client sample: WSClient (..\Program Files\SAP\SAP Business One SDK\Samples\DI Server/DIServerBP/WSClient)
  4. Run the Client sample by opening “WSClient.vbproj” and run the project.
