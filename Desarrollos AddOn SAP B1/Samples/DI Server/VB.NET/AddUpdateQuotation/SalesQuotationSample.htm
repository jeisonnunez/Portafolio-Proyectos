1) DI Server – SalesQuotation Sample

1.1) Configuration

Sample requirements:

You need the following software installed:

  1. IIS Server.
  2. .Net framework
  3. Visual Studio .Net 2003

 

In order to use this sample follow these steps:

  1. Copy the folder “wsSalesQuotation” (it’s in the same folder with this file) to your IIS folder, usually “C:\interpub\wwwroot”.
  2. Open the “Internet Services Manager” under Start-> Programs-> Administrative tools.
  3. Open your server node, and the default web server node.
  4. Locate the “wsSalesQuotation” folder, right-click on it and select properties.
  5. Click the “Create” button circled in the following image.

 

 

  1. Open the file “wsDiServerServices.vbproj”
  2. Build the project and run it.
  3. Open the Client sample: SalesQuotationClient (..\Program Files\SAP\SAP Business One SDK\Samples\DI Server/AddUpdateQuotation/SalesQuotationClient)
  4. Run the Client sample by opening “SalesQuotationClient.vbproj” and run the project.

 

1.2) Running the sample

The sample uses the “SBODemo_US” demo.

To use this sample follow these steps:

1)     Log In to company :

Check that the log in parameters are valid.

2)     Add Quotation :

Use BP code and item code that already exists in the database.

3)     Update :

Updates the first item in the Quotation.

The update uses the “Doc No” of the added Quotation.

Insert an item code and quantity.

4)     Log out :

Log out when you finish.
